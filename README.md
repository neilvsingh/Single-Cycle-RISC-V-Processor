# Single-Cycle-RISC-V-Processor
Developed a Verilog based computer processor capable of executing simple RISC-V programs. 
Processor includes 1028 bits of data memory, a RISC-V instruction decoder, an arthimetic logic unit, 
and a program counter module. The CPU is able to execute simple RISC-V programs supporting branching.
Processor was implemented on the TUL Embedded PYNQ board using Xilinx Vivado.
